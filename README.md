# M1 Modern JavaScript Lab

Frontend responsibilities focus on what the user sees and interacts with, such as layout, buttons, forms, and client side behavior. Backend responsibilities focus on server side logic, databases, authentication, APIs, and handling data securely.

The request response lifecycle starts when a client, such as a browser, sends a request to a server. The server processes the request, may interact with a database, and sends a response back to the client. Modular file organization is important because it keeps code easier to read, maintain, test, and reuse as projects grow.
